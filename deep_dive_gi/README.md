## 核心观点

1. **高度产品化的运行时**：Claude Code 不仅仅是一个 API 客户端，它是一套完整的 Agent 运行时，通过复杂的 `query.ts` 状态机管理异步工具调用。
2. **UI 与逻辑深度耦合**：使用 React + Ink 构建 TUI，但 UI 层（如 `REPL.tsx`）承担了大量会话协调逻辑。
3. **多级上下文治理**：面对超长对话，系统通过 `microcompact`、`autocompact` 等多级策略精细化管理 token。
4. **扩展性设计**：通过 MCP（Model Context Protocol）和内部的 `skills` 系统，实现了能力的高度可插件化。
